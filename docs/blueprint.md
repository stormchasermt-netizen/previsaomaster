# **App Name**: PRevisão Master

## Core Features:

- Real-time Data Ingestion: Collect and process weather data from various sources in real-time.
- Forecast Generation: Generate weather forecasts using collected data and AI algorithms to predict weather patterns. The AI can use external weather APIs and historical data as tools to create the best prediction.
- Interactive Map Display: Display weather information on an interactive map, showing temperature, precipitation, wind speed, and other relevant data.
- Customizable Alerts: Allow users to set up custom alerts for specific weather conditions, such as high winds, heavy rain, or temperature changes.
- Hourly and Daily Summaries: Provide hourly and daily weather summaries with key insights and potential impacts.
- Historical Weather Data: Access historical weather data for a specific location and time period.
- Location Search: Search weather information using geolocation or direct search by entering the name of city.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to reflect the app's theme.
- Background color: Light gray (#F0F8FF), almost white, for a clean interface.
- Accent color: Warm yellow (#FFD700) to highlight key information and interactive elements.
- Body and headline font: 'Inter', a sans-serif font for a clean, modern look, is suitable for both headlines and body text.
- Use clear, weather-related icons that are easy to understand.
- Clean and intuitive layout with a focus on readability and accessibility.
- Smooth transitions and subtle animations to enhance user experience.
